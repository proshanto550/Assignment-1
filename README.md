# 📘 Problem Solutions

## Problem 1
Create a TypeScript function `filterEvenNumbers` that accepts an array of numbers and returns only the even numbers.

## Problem 2
Write a function `reverseString` that reverses a string.

## Problem 3
Define a union type `StringOrNumber` and create a function `checkType` using type guards.

## Problem 4
Create a generic function `getProperty` that safely returns object properties using constraints.

## Problem 5
Define a `Book` interface and create a function `toggleReadStatus` that adds an `isRead` property.

## Problem 6
Create a `Person` class and a `Student` subclass with a `getDetails()` method.

## Problem 7
Create a function `getIntersection` that returns common elements between two arrays.
